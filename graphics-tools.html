Graphics Tools


  • BlackOutV2.02

    This is a screensaver frontend with lots of different savers and a random mode. RISC OS 4 and later has its own Screensaver system.

  • SpriteLibV2.14

    Application which moves sprites with the same name into a single sprite file to make it easy to compare them.

  • FatPaintV0.02a

    A pixel oriented drawing program by Mark Greenwood.

  • InsertIconV1.05

    Inserts sprites into the standard sprite pool, RISC OS Select priority sprite pool or replaces toolsprites.

  • GermsV3.04

    Module which replaces the startup banner with a picture (Sprite, JPEG, Draw) of your choice. Versions prior to v3.00 were developed by Mark Greenwood and part of the BlackHole application.
